 🛒 Econverse

Projeto desenvolvido com React + TypeScript utilizando Vite.

 🚀 Tecnologias

- React
- TypeScript
- Vite
- CSS

 📦 Requisitos
Antes de começar, você precisa ter instalado:
- Visual Studio Code
- Node.js (versão 18 ou superior)
- npm (geralmente já vem com o Node)
Para verificar:
(crtl + shift + `) para abrir um novo terminal no vscode
  bash
node -v
npm -v
Entre na pasta:
cd nome-do-projeto

  📦 Instalando as dependências
npm install

  ▶️ Rodando o projeto em ambiente de desenvolvimento
npm run dev
O projeto estará disponível em:
http://localhost:5173 se não vai aparecer um link no terminal, só apertar crtl + botão esquerdo em cima do link.

  🏗 Gerando build para produção
npm run build

  🔎 Visualizando build localmente
npm run preview
